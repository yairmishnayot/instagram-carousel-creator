I want to create an Hebrew-first web application that will help a user create instagram carousels from given textual content.

- The concept is that the user will put textual content into an input, and that content will be transformed into a beautiful slide.
- The app must support RTL as default language, no need for other languages for now.
- All the images created with the app has to be in a size that is good for an instagram post.
- The user will be able to add more slides as much as he wants.
- The user will be able to remove slides as well.
- by default, have 10 different color pallets available to choose from. take them from https://coolors.co/palettes/trending
- After clicking to generate carousel, there will be a preview of all the slides.
- The user should be able to edit the content of the slides, and also change design aspects and styling of the slides.
- The user should be able to give a main title for the carousel.
- The user should be able to export all of the carousel images and download them. When doing so, name each image with the carousel title as a prefix, and then add a number based on the order of the image.
- The user should also be able to download individual carousel images.
